Moss - An IRC bot written in Go
===============================

Moss is an IRC bot written in Go. It is based on the `go-ircevent
<https://github.com/thoj/go-ircevent>`_ library.


Features
--------

- Respond to mentions


Goals
-----

- Plugin support
